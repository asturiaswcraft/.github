# Os damos la bienvenida a Asturias Software Crafters! 👋🏻

Grupo de personas entusiastas de la tecnología, que quieran juntarse para compartir experiencias y conocimientos.

Organizaremos charlas, talleres y crearemos un espacio seguro donde pasar un rato divertido.

Nuestra misión es generar un punto de encuentro para aprender y estar en comunidad.
